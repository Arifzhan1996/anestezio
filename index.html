🩺 АнестезиоЛог — Настройка

Для синхронизации между врачами из разных городов нужно подключить бесплатную облачную базу данных. Это делается один раз.

1
Перейдите на jsonbin.io и нажмите Sign Up — зарегистрируйтесь бесплатно
2
После входа нажмите на своё имя (вверху справа) → API Keys → скопируйте ключ под надписью Secret Access Key
3
Вставьте ключ ниже и нажмите «Подключить». После этого раздайте этот HTML-файл всем врачам — они будут работать с одной базой.
